Tipos de Operadores Lógicos em JavaScript

O operador AND (&&) retorna true se as 2 expressões são verdadeiras (true), caso contrário, retorna falso (false).

O operador OR (||) retorna true se 1 ou 2 expressões são verdadeiras (true), caso contrário, retorna falso (false).

O operador NOT (!) retorna true para expressões falsas e false para expressões verdadeiras.
